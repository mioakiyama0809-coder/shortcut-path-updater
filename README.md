# Shortcut Path Updater

A simple Windows tool to fix broken Start Menu shortcuts after moving software folders.

## Usage
1. Download `UpdateShortcut.exe`
2. Run as Administrator
3. Step 1: Browse and select the broken `.lnk` shortcut file
4. Step 2: Browse and select the new `.exe` location
5. Click **Update Shortcut**

## Requirements
- Windows 10 / 11
